#Description

Ce projet est un **jeu rogue-like simple** dans lequel le joueur doit **tuer le plus d’ennemis possible**.  
Le but est de survivre le plus longtemps possible tout en explorant différentes salles et en affrontant des vagues d’adversaires.

---

#Contrôles du jeu

| Touche | Action |
|:-------|:--------|
| ⬆️ / ⬇️ / ⬅️ / ➡️ | Se déplacer |
| **E** | Attaquer |
| **Q** | Se téléporter entre les salles |

---

#Objectif

- Explorer les salles générées aléatoirement  
- Affronter et éliminer un maximum d’ennemis  
- Utiliser la téléportation pour avancer entre les salles
